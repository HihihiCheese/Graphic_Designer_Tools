# 工作日志 2026-03-01

## 今日完成

### Phase 2：排版生成工具 (tool2_layout) — 全部完成

从秀米剪贴板抓取的真实模板 HTML 出发，搭建了完整的排版生成工具。

#### 新建文件
- `tool2_layout/components.py` — 从秀米模板提取的 ~35 个 HTML 组件函数（6 套风格）
- `tool2_layout/renderer.py` — 渲染引擎，JSON → HTML
- `tool2_layout/main.py` — CLI 入口
- `tool2_layout/templates/news.html` — 蓝色模板1（新闻稿）
- `tool2_layout/templates/news_blue2.html` — 蓝色模板2（短稿）
- `tool2_layout/templates/news_red.html` — 红色模板1（党建复杂版）
- `tool2_layout/templates/news_red2.html` — 红色模板2（党建简约版）
- `tool2_layout/templates/news_cyan.html` — 青色模板（学术论坛）
- `tool2_layout/templates/news_purple.html` — 紫色模板（正式通知）
- `tool2_layout/templates/lecture.html` — 讲座模板

#### 核心设计决策
1. **不从零写 CSS，复用秀米 HTML 片段**：直接把秀米模板拆成可复用组件（text_card、image_frame、decorator_line 等），保留原始内联样式和 CDN 图片 URL
2. **7 套模板**：根据 `--template` 参数或 `category` 字段选择模板
3. **所有 CSS 内联**：微信公众号会删除 `<style>` 标签，所以全部写成 inline style
4. **讲座默认署名**：讲座模板在 credits 为空时自动填充美编/责编占位符
5. **红色模板图片智能样式**：连续 3+ 张图片才启用丝带/偏移轮换样式，单独图片一律居中

#### 遇到的问题与修复
- **图片路径重复拼接**：`images_dir` 从 `extracted.json` 读取时已包含 `output/xxx/images` 前缀，再与 json 所在目录拼接会导致路径重复。修复为优先从 cwd 解析，兜底用 json 同目录下的 images/
- **浏览器无法打开含空格路径**：`webbrowser.open()` 在 Windows 上对 `file:///` URI 处理有问题，改用 `os.startfile()` 直接打开
- **测试 JSON 中的中文引号**：中文双引号 `""` 在 JSON 字符串内破坏解析，需要用 `\u201c\u201d` 转义

#### 测试验证
- 用 "a talk.docx" 讲座文档完成端到端测试（tool1 提取 → 手动修正 JSON → tool2 渲染 → 浏览器预览）
- 渲染效果：头尾图 GIF、蓝线+黄点装饰线、信息区、虚线框内容卡片、报告人头像、署名区均正常显示

### 模板扩展 — 新增 5 套通用模板

从秀米剪贴板逐一抓取真实模板 HTML，提取组件并创建 Jinja2 模板。

#### 新增模板一览

| 模板名 | 文件 | 风格 | 适用场景 |
|--------|------|------|----------|
| `news_blue2` | `templates/news_blue2.html` | 深蓝边框 + 浅蓝底 + 折角装饰 | 较短的新闻稿/活动稿 |
| `news_red` | `templates/news_red.html` | 红色双线/虚线 + 折角卡片 + 丝带图片 | 党建活动（复杂版） |

| `news_purple` | `templates/news_purple.html` | 紫色渐变标题栏 + 浅紫背景 + 方块装饰 | 正式通知/会议 |

#### 新增组件（components.py）

- **蓝色模板2**：`blue2_frame_open/close`, `blue2_image`, `blue2_paragraph`
- **红色模板1**：`red_separator`, `red_dashed_line`, `red_text_card`, `red_activity_banner`, `red_numbered_heading`, `red_text_block`, `red_image`, `red_image_with_ribbon`, `red_image_with_bar`, `red_credits`
- **红色模板2**：`red2_frame_open/close`, `red2_paragraph`, `red2_image`, `red2_diamond_separator`, `red2_credits`
- **青色模板**：`cyan_text_card`, `cyan_numbered_heading`, `cyan_text_block`, `cyan_image`, `cyan_credits`
- **紫色模板**：`purple_heading`, `purple_text_block`, `purple_image`, `purple_separator`, `purple_credits`

#### 红色模板图片智能样式

- `_mark_image_groups()` 函数：检测连续 3 张及以上图片，标记为 `in_group=True`
- 模板中：`in_group` 图片使用丝带/偏移条/居中三种样式轮换；单独图片一律居中 `red_image()`

#### 模板强调色映射

- 蓝色系模板（news, news_blue2, news_cyan, news_purple, lecture）→ `rgb(32, 90, 171)`
- 红色系模板（news_red, news_red2）→ `rgb(185, 27, 21)`
