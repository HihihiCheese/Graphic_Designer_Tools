### ImgBB 图床支持

SM.MS 图床需付费，改用免费的 ImgBB（单张≤32MB，无限存储）。

#### 新增/修改文件
- **image_hosting.py**：新增 `ImgbbImageHost` 类，base64 编码上传，与 `SMImageHost` 同接口
- **config.py**：新增 `get_image_hosting_config()` 返回 `(provider, token)`，支持 `imgbb`/`smms` 切换
- **main.py**：根据 `provider` 字段自动创建对应图床客户端
- **config.example.json**：更新为 ImgBB 优先的配置模板

#### 配置方式
```json
{
  "image_hosting": {
    "provider": "imgbb",
    "token": "ImgBB API Key"
  }
}
```

### CDN 素材图片上传

发现秀米 CDN 的头尾图/装饰 GIF 在公众号后台复制粘贴时无法正常显示。

#### 解决方案
- **html_processor.py**：新增 `_find_cdn_images()` + `_download_and_upload()` 函数
- 处理流程：先替换本地 `file:///` 图片 → 再查找外部 CDN 图片 → 下载到临时文件 → 上传图床 → 替换 URL
- 智能跳过已上传的图床 URL（ibb.co、smms 等），避免重复上传

### Bug 修复
- **tool3 main.py**：修复 `publish_helper.html` 写入时的 `UnicodeEncodeError`（surrogate 字符），添加 `errors="replace"`
